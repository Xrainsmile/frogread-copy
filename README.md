# ReadFlow — 沉浸式双语阅读翻译 `v1.0 正式版`

> 一键翻译网页，原文译文对照阅读，支持 38 种语言、多家 AI 模型（混元 / DeepSeek / 智谱 / OpenAI / 硅基流动等）。
>
> **零基础也能用**：选一个模型 → 填一个 Key → 点「测试连接」→ 开始翻译。下面手把手教你配置。

---

## 🚀 30 秒快速上手

### 第一步：安装扩展

1. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions`
2. 打开右上角的 **「开发者模式」** 开关
3. 点击 **「加载已解压的扩展程序」**，选择 `dist` 文件夹
4. 扩展栏会出现 ReadFlow 图标 ✅

### 第二步：配置 API（最关键的一步，跟着做就行）

点击浏览器右上角扩展栏的 **ReadFlow 图标**，打开设置面板。你会看到一个「**API 提供商**」下拉框，里面分成两组：

```
腾讯内部
  ├ 混元 (Hunyuan)
  └ 太极 (Taiji)
第三方 / 开源模型（OpenAI 兼容）
  ├ DeepSeek
  ├ 智谱 GLM
  ├ OpenAI
  ├ SiliconFlow
  └ 自定义 OpenAI 兼容 API
```

**你只需要做两件事：① 选一个提供商　② 填上 API Key。** 其它（地址、模型）都会自动填好。

**方式一：使用公司内部 API**

| 设置项 | 填写内容 |
|--------|---------|
| API 提供商 | 选择 `混元 (Hunyuan)` 或 `太极 (Taiji)` |
| API Key | 填入你的 API Key |
| 翻译模型 | 默认即可（旗舰 Pro 模型），一般不用改 |

**方式二：使用外部 API（DeepSeek、智谱、OpenAI、硅基流动等）👈 个人用户推荐**

| 设置项 | 填写内容 |
|--------|---------|
| API 提供商 | 直接选 `DeepSeek` / `智谱 GLM` / `OpenAI` / `SiliconFlow` |
| API Key | 填入你从对应平台获取的 Key（一般以 `sk-` 开头） |

> 💡 选好提供商后，「**API 地址**」和「**模型名称**」会**自动填好**，你只管填 API Key 即可！
> 如果你用的是上面没列出的服务，选最后一项 `自定义 OpenAI 兼容 API`，手动填地址和模型名称就行。

**③（强烈建议）点一下「测试连接」按钮**

填好后，先点 **「测试连接」** 按钮验证一下：

- ✅ 显示绿色 **「连接成功」** → 配置没问题，可以用了
- ❌ 显示红色错误提示 → 按提示修改，常见错误见下表：

| 红色提示包含 | 说明 | 怎么解决 |
|--------------|------|----------|
| `认证失败 (401)` / `无访问权限 (403)` | **API Key（Token）填错了**或已过期 | 重新复制粘贴正确的 Key |
| `地址/模型错误 (404)` | API 地址或模型名称不对 | 换回预设，或检查地址/模型拼写 |
| `无法连接到 API 地址` | 网络不通或地址写错 | 检查网络（公司 API 需连 VPN）、确认地址正确 |
| `请求过于频繁 (429)` | 触发了限流 | 稍等片刻再试 |

确认连接成功后，点击 **「保存设置」**，看到绿色的「已保存」提示就大功告成了 🎉

如果你不想花一分钱就能用上 AI 翻译，推荐使用 **硅基流动（SiliconFlow）** 平台，它提供 **完全免费** 的 Qwen2.5-7B-Instruct 模型。

1. 打开 [cloud.siliconflow.cn](https://cloud.siliconflow.cn)
2. 用**手机号**或**微信扫码**注册/登录

1. 登录后，进入 [API 密钥页面](https://cloud.siliconflow.cn/account/ak)
2. 点击 **「新建 API 密钥」**
3. 复制生成的密钥（以 `sk-` 开头），保存好

#### 第 3 步：在 ReadFlow 中配置

点击扩展栏的 ReadFlow 图标，按下表填写：

| 设置项 | 填写内容 |
|--------|---------|
| API 提供商 | 直接选 **`SiliconFlow`**（地址和模型会自动填好 ✨） |
| API Key | 粘贴你刚复制的 `sk-...` 密钥 |

> 💡 选择 SiliconFlow 后，**API 地址**和**模型名称**会自动填好，你只需要填 API Key 就行！
>
> 自动填入的值：
> - API 地址：`https://api.siliconflow.cn/v1/chat/completions`
> - 模型名称：`Qwen/Qwen2.5-7B-Instruct`

填完后先点 **「测试连接」** 确认显示绿色「连接成功」，再点 **「保存设置」**，然后打开一个英文网页试试翻译吧 🎉

#### ⚠️ 免费模型须知

| 项目 | 说明 |
|------|------|
| 💰 费用 | **完全免费**，无需充值 |
| 🚀 速度 | 有一定的速率限制，翻译大量内容时可能稍慢 |
| 📏 上下文 | 32K tokens，足够翻译绝大多数网页段落 |
| 🌐 网络 | 需要能访问外网（不需要 VPN，国内直连即可） |

> 💡 如果觉得免费模型速度不够或翻译质量想要更好，可以随时切换到 DeepSeek 等付费方案（费用也很低，几块钱能用很久）。

### 第三步：开始翻译！

打开任意英文网页（比如 [Hacker News](https://news.ycombinator.com)），然后：

- **方法 1**：点击页面右下角的 **蓝色悬浮按钮** 🔵
- **方法 2**：按快捷键 **⌥T**（Mac）或 **Alt+Shift+T**（Windows）
- **方法 3**：点击扩展栏图标，点击 **「翻译」** 按钮

翻译完成后，每段原文下方会出现对应的中文译文 🎉

## 📖 功能详解

### 翻译网页

| 操作 | 说明 |
|------|------|
| 🔵 **点击悬浮按钮** | 页面右下角蓝色圆形按钮，点一下开始翻译 |
| 🟢 **翻译完成** | 按钮变绿，表示翻译已完成 |
| 🔵🔵 **双击悬浮按钮** | 关闭翻译，恢复原始页面 |
| 🟡 **按钮变黄** | 翻译进行中，请稍等 |
| 🔴 **按钮变红** | 翻译出错，页面右上角会弹出红色提示说明具体原因（详见常见问题） |

> 💡 **增量翻译**：翻译后如果页面加载了新内容（比如滚动加载更多评论），再次点击悬浮按钮，会**只翻译新内容**，不会重复翻译已有的部分。

### 三种显示模式

在设置面板的「显示模式」中切换：

| 模式 | 效果 |
|------|------|
| **双语对照**（默认） | 原文和译文同时显示，对照阅读 |
| **仅译文** | 只显示翻译结果，隐藏原文 |
| **仅原文** | 隐藏译文，恢复原始页面显示 |

### 划词查询

按住 **Alt（⌥）键**，将鼠标悬停在英文单词上，会弹出一个小气泡显示中文释义。松开 Alt 键气泡自动消失。

### 多配置管理

如果你有多个 API Key（比如公司内部的和个人的），可以创建多个配置：

1. 点击 API 配置旁的 **＋** 按钮新增配置
2. 给配置起个名字（如「公司混元」「个人 DeepSeek」）
3. 填写对应的 API 信息并保存
4. 通过下拉框快速切换不同配置

## ❓ 常见问题

### Q: 不确定 API 配置对不对？

打开扩展面板，点 **「测试连接」** 按钮即可：绿色「连接成功」表示没问题；红色提示会**直接告诉你哪里错了**（API Key 错、地址错、网络不通等），照着提示改就行。建议每次配置后都先测一下再保存。

### Q: 点击翻译没反应？

1. 检查是否已保存 API Key（打开扩展面板确认），可点「测试连接」验证
2. 如果使用公司内部 API，确保已连接 VPN / 在办公网络内
3. 尝试刷新页面后重试

### Q: 翻译按钮变红了 / 页面右上角弹出红色提示？

说明 API 调用失败。**页面右上角会弹出一个红色提示框，写明具体原因**，常见的有：

| 提示包含 | 原因 | 解决办法 |
|----------|------|----------|
| `认证失败 (401)` / `无访问权限 (403)` | API Key 填错或过期 | 重新粘贴正确的 Key |
| `地址/模型错误 (404)` | 地址或模型名称不对 | 换回预设或检查拼写 |
| `无法连接到 API 地址` | 网络不通 / 地址错 | 检查网络（公司 API 需连 VPN） |
| `请求过于频繁 (429)` | 触发限流 | 稍等片刻再试 |

> 💡 拿不准时，点扩展面板的「测试连接」按钮，能更快定位问题。

### Q: 翻译结果不准确？

可以尝试切换不同的模型/API 提供商。不同模型的翻译风格有差异：
- **SiliconFlow（Qwen2.5-7B）**：免费，翻译质量不错，适合日常阅读
- **DeepSeek**：性价比高，翻译自然流畅
- **GLM**：中文表达地道
- **GPT**：翻译质量稳定

### Q: 怎么只翻译页面的一部分？

目前 ReadFlow 会翻译页面上检测到的所有正文段落。代码块、导航栏、按钮等非正文内容会被自动跳过。

### Q: 支持 PDF 翻译吗？

✅ **支持**。在浏览器里打开 PDF 文件，点击扩展面板的 **「翻译」** 按钮，页面右侧会弹出一个翻译面板，逐页显示原文与中文译文对照。

## 📁 项目结构（开发者参考）

```
dist/               ← Chrome 加载这个文件夹
src/
  ├── manifest.json       ← 扩展配置
  ├── popup/              ← 弹出面板（设置界面）
  ├── content/            ← 内容脚本（注入网页、检测段落、注入译文）
  ├── background/         ← 后台脚本（API 调用、缓存、消息路由）
  └── shared/             ← 共享类型定义
```
